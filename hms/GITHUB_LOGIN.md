# GitHub Login Instructions

## Step 1: Open Command Prompt
- Press `Win + R` on your keyboard
- Type `cmd` and press Enter

## Step 2: Run GitHub Login Command
In the command prompt window, type:
```
gh auth login
```

## Step 3: Follow the Prompts (Type the numbers and press Enter)

**Q1: What account do you want to log into?**
```
1. GitHub.com
2. GitHub Enterprise Server
```
**Type:** `1` and press Enter

**Q2: What is your preferred protocol for Git operations?**
```
1. HTTPS
2. SSH
```
**Type:** `1` and press Enter

**Q3: How would you like to authenticate GitHub CLI?**
```
1. Login with a web browser
2. Paste an authentication token
```
**Type:** `1` and press Enter

**Q4: Press Enter to open github.com device activation page...**
- Just press Enter

## Step 4: Complete in Browser
- A browser page will open
- You'll see a code like `XXXX-XXXX` (example: ABCD-1234)
- Copy that code
- Paste it in the browser window
- Click "Continue"

## Step 5: Authorize
- Click "Authorize github"

## Step 6: Come back here
Once done, tell me "Done" and I'll push your project to GitHub!
